NumeriKet

A collection of Racket modules implementing numerical methods.

1 Implemented Functions

1.1 Root Finding

procedure

(newton-root f x0) → number?

  f : procedure?
  x0 : number?
Approximate the root of f closest to x0.

This is implemented through Newton’s method for finding roots of real valued functions and uses ten iterations. Note that f must have a continuous second derivative.

procedure

(newton-sqrt s x0) → number?

  s : number?
  x0 : number?
Approximate the square root of s by starting at x0.

This is implemented through a specific case of newton-root and uses ten iterations.

1.2 Derivatives

procedure

(diff f x0 [#:method method]) → number?

  f : procedure?
  x0 : number?
  method : string? = "newton-diff"
Estimate the value of the derivative of f, a function of x, at the point x0, using the method method.

By default, five-point is used.

procedure

(five-point f x0) → number?

  f : procedure?
  x0 : number?
Estimate the value of the derivative of f, a function of x, at the point x0.

This is implemented through the five point method with a step size of 1e-7.

procedure

(newton-diff f x0) → number?

  f : procedure?
  x0 : number?
Estimate the value of the derivative of f, a function of x, at the point x0.

This is implemented through Newton’s difference quotient with a step size of 1e-7.

1.3 Integrals

procedure

(integrate f int [#:method method]) → number?

  f : procedure?
  int : list?
  method : string? = "simpson"
Estimate the value of integrating f over the interval int.

By default, simpson is used.

procedure

(simpson f int) → number?

  f : procedure?
  int : list?
Estimate the value of integrating f over the interval int.

This is implemented through Simpson’s rule. By default, int is partitioned into intervals of length 0.001 up to a maximum of 50000 intervals. If the length of int is greater than 50, then larger intervals are used to partition the interval into 50000 intervals.

1.4 Solving ODEs

procedure

(euler-method f inits tf) → number?

  f : procedure?
  inits : list?
  tf : number?
Given inits of the form '(t0 x0), approximate the value of the solution of f at tf, where (x t0) is x0.

This is implemented through Euler’s method for solving first order differential equations, and uses a step size of 0.0001. Note that f, the flow of x in time, is a function of t and x (in that order).
